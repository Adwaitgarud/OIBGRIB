Hyper Text Markup Language (HTML)

It is used to tell your web browser what each part of a website is. So, using HTML, you can define headers, paragraphs, links, images, and more, so your browser knows how to structure the web page you're looking at.

Cascading Style Sheets (CSS)

CSS helps you to keep the informational content of a document separate from the details of how to display it. The details of how to display the document are known as its style.

JavaScript (JS)

JavaScript is a programming language that adds interactivity to website. This happens in games, in the behavior of responses when buttons are pressed or with data entry on forms; with dynamic styling; with animation.

Object Oriented Programming (C++)

C++ is a cross-platform language that can be used to create high-performance applications. C++ was developed by Bjarne Stroustrup, as an extension to the C language. C++ gives programmers a high level of control over system resources and memory.
